Creating a tool for automating sorting of objects containing the following sorts:
1. Selection sort
2. Bubble sort
3. Insertion sort
4. Merge sort
5. Quick sort
6. Heap sort
7. Comb sort 
Instructions 
 Input a number to determine your list’s length.
The program creates a list of random natural numbers.
 Choose a sort within the sorts available.
 The program :
 sorts the list 
 displays the ordered list in the terminal
 returns the length of the list, the name of the sorting method and the execution time in a table
 displays a pie graph of the ordered list
